# Feynman Learning Journal

> 我把学到的编程知识，**用自己的话讲给“完全新手”**。讲不清的地方就是盲点——回去补、再讲、直到讲清为止。  
> 方法：**费曼学习法（Feynman Technique）**

---

## 为什么叫 “Feynman”？
致敬物理学家 **Richard P. Feynman（理查德·费曼）** 提出的学习思路：  
解释 → 发现盲点 → 补齐 → 再解释 → 简化。  
本仓库就是我应用这套方法的**长期笔记与复盘**。

---

## 我怎么用这个仓库？
- **写**：每学到一个概念/技巧，就新建一篇 Markdown 按模板写。
- **讲**：假设读者是第一次接触该主题，用**通俗语言 + 案例**讲给“10 岁的自己”。
- **迭代**：发现讲不清的地方在文末开「盲点清单」，补齐后更新版本记录
